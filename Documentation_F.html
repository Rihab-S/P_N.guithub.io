Méthode Multi-modale pour Prédiction des Notes

Concept

Le modèle combine différentes sources d'information : texte (titre), catégorielle (ASIN), numérique (prix), et image (photo). Chaque type de donnée est transformé en vecteurs de caractéristiques, puis tous sont concaténés pour former une représentation complète du produit.

Sources de données et transformations

  • Texte (titre) : Transformé en vecteurs TF-IDF pour capturer la signification des mots.
  • Catégorie (ASIN) : One-hot encoding pour capturer l’identité unique du produit.
  • Prix : Normalisation après imputation pour capturer l’effet du prix.
  • Image : ResNet18 pré-entraîné génère 512 caractéristiques représentant l’apparence visuelle.

Combinaison multi-modale

Toutes les caractéristiques sont concaténées pour former un vecteur unique par produit :

X_final = [TF-IDF titre | One-hot ASIN | Prix normalisé | Features image]

Ce vecteur est ensuite utilisé comme entrée pour le modèle CatBoost qui prédit la note.

Pourquoi cette approche est efficace

  • Multi-modale : Utilise toutes les sources d'information pour une meilleure prédiction.
  • TF-IDF + ASIN : Capture le texte descriptif et l'identifiant unique du produit.
  • Prix + Image : Capture les indices numériques et visuels influençant la perception.
  • CatBoost : Modèle performant pour données hétérogènes, capable de combiner numérique et catégoriel.

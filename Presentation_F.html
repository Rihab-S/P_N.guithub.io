Présentation MVP - Ponera

Cette page présente le MVP proposé pour anticiper le risque de retour des produits vendus sur Amazon.

Objectif

Le but de ce MVP est de proposer une solution basée sur l’intelligence artificielle pour prédire les produits à fort risque de retour avant leur mise en vente.

Défi

Le défi consiste à :

  • Identifier les produits à risque sur Amazon (mention “Vendu par Ponera”).
  • Récupérer leur ASIN et utiliser les données disponibles pour calculer un score de risque de retour.
  • Présenter un prototype ou une démonstration de votre solution IA.

Approche proposée

Le MVP peut inclure :

  • Un modèle IA simple pour prédire le risque de retour.
  • Une visualisation des produits avec leur score de risque.
  • Une démonstration basée sur les données disponibles pour chaque ASIN.

L’objectif est de montrer comment la solution peut réduire les retours coûteux et améliorer la gestion des produits.
